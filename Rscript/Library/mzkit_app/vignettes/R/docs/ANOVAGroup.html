ANOVAGroup {mzkit} R Documentation

Create a dataset for evaluate ANOVA p-value

Description

Usage

ANOVAGroup(data, sampleinfo);

Arguments

sampleinfo

a list data that should contains the data sample group tag information. the data structre of this list could be two fields are required at least: + group: the sample group name + id: a character vector that contains the sample id to get intensity vector data from the data parameter.

Details

Authors

xieguigang

Value

a dataframe object that contains two fields: intensity and region_group, which could be used for do stat chartting plot liked bar/box/violin and also could be used for evaluated ANOVA F-test p-value result.
