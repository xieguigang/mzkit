ms1_peaktable {mzkit} R Documentation

Export peakstable data from the XIC rawdata files

Description

Usage

ms1_peaktable(files, mzbins,
    mzdiff = 0.005,
    peak.width = [3, 90]);

Arguments

files

a character vector of the file path of the XIC rawdata files

mzbins

A dataframe object that contains the target ion m/z feature set for do extract of the result peaktable data, a data field which is named mz must be included inside this dataframe object.

peak.width

the peak time range of the peaks that could be accepted

Details

Authors

xieguigang

Value

this function generates a xcms format liked peaktable dataframe object for the input rawdata files.
