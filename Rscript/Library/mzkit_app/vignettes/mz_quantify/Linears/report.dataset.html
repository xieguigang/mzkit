report.dataset {Linears} R Documentation

Create targeted linear dataset object for do linear quantification data report.

Description

Usage

report.dataset(standardCurve, samples,
    QC.dataset = NULL,
    ionsRaw = NULL);

Arguments

standardCurve

[as StandardCurve]

samples

[as QuantifyScan]

QC.dataset

Regular expression pattern string for match QC sample files. [as string]

ionsRaw

A list of ions ChromatogramTick data for run ion data plots. the list data structure in format looks like: 

 { 
    ion_id1 {
        sample1: ChromatogramTick[],
        sample2: ChromatogramTick[],
        ...
    }
 }
. [as list]
