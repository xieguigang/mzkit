from_arguments_json {MRMLinear} R Documentation

Create the MRM peak finding arguments from a json string

Description

Usage

from_arguments_json(json.str,
    parse.single = TRUE);

Arguments

json.str

[as string]

parse.single

this function will try to parse the given json string as MRMArguments if `parse_single` is set to true, otherwise a set of the ion parameters MRMArgumentSet will be parsed if `parse_single` is set to false. [as boolean]
