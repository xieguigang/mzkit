parse_xcms_samples {xcms} R Documentation

Parse the input file as the mzkit peakset object

Description

Usage

parse_xcms_samples(file,
    group.features = FALSE,
    deli = ",",
    normalizeID = TRUE);

Arguments

file

-

group.features

This function returns a xcms sample peaks collection for directly mapping of the input tabular file data. set this parameter to value true, will returns a tuple list object that contains the mzkit peak feature groups, which is group by the sample names. [as boolean]

env

[as Environment]

Details

Authors

BioNovoGene

Value

this function returns data object in these one of the listed data types: XcmsSamplePeak, PeakFeature.

clr value class

  • PeakFeature
