conservedGuid {spectrumPool} R Documentation

generates the guid for the spectrum with unknown annotation

Description

Usage

conservedGuid(spectral);

Arguments

spectral

-

Details

the conserved guid is generated based on the md5 hashcode of contents: 1. mz(F4):into 2. mz1(F4) 3. rt(F2) 4. biosample 5. organism 6. instrument 7. precursor_type
