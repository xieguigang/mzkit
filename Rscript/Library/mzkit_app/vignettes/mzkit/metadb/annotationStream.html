annotationStream {metadb} R Documentation

Construct a basic metabolite annotation data collection

Description

Usage

annotationStream(id, name, formula);

Arguments

id

the unique reference id of the metabolite collection. [as string]

name

common names. [as string]

formula

metabolite molecular formula data collection. [as string]

env

[as Environment]

Details

the exact mass will be evaluated based on the input `formula` data.

Authors

BioNovoGene

Value

this function returns data object of type MetaboliteAnnotation.

clr value class

  • MetaboliteAnnotation

Examples

 annotationStream(
     id = ["met1","met2","met3"],
     name = ["name1", "name2", "name3"],
     formula = ["CH3OH","CH3OOOOOH","CH4NH4"]
 );
