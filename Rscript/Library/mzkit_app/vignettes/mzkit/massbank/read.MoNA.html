read.MoNA {massbank} R Documentation

read MoNA database file.

Description

Usage

read.MoNA(rawfile,
    skipSpectraInfo = FALSE,
    is.gcms = FALSE,
    lazy = TRUE,
    verbose = TRUE);

Arguments

rawfile

a vector of the mona database file, could be a set of multiple mona database file. the database reader is switched automatically based on this file path its extension name. currently supported data file formats: sdf and msp. [as string]

is.gcms

Load gcms reference dataset?. [as boolean]

lazy

Create a lazy data populator or load all data in memory and returns a vector of the spectral data. [as boolean]

env

[as Environment]
