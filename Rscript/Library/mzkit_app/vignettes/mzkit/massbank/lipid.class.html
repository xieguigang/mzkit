lipid.class {massbank} R Documentation

Create lipid class helper for annotation

Description

Usage

lipid.class(lipidmaps,
    id = NULL);

Arguments

lipidmaps

-

env

[as Environment]

Details

Authors

BioNovoGene

Value

this function returns data object in these one of the listed data types: LipidClassReader, CompoundClass.

clr value class

  • LipidClassReader
  • CompoundClass

Examples

 # cast sdf dataset to lipidmaps data object
 let dataset = read.SDF(file = "./example.sdf", lazy = FALSE);
 let lipids = dataset |> as.lipidmaps();
 
 # create annotation helper
 let class = lipid.class(lipids);
