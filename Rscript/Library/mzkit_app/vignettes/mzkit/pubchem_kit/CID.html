CID {pubchem_kit} R Documentation

query cid from pubchem database

Description

Usage

CID(name,
    cache = "./.pubchem",
    offline = FALSE,
    interval = -1);

Arguments

name

any search term for query the pubchem database. [as string]

cache

the cache fs for the online pubchem database

offline

running the search query handler in offline mode?. [as boolean]

interval

the time sleep interval in ms. [as integer]

Details

Authors

BioNovoGene

Value

A character vector of the pubchem cid that matches the given input name.
