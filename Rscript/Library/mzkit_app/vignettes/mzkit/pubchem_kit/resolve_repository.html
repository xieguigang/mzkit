resolve_repository {pubchem_kit} R Documentation

load pubchem local repository

Description

Usage

resolve_repository(repo);

Arguments

repo

a directory path to the local pubchem repository. [as string]

Details

Authors

BioNovoGene

Value

a collection of the pubchem PugViewRecord data, which could be converted to the mzkit internal metabolite metadata annotation model via the function metadata.pugView.

clr value class

  • PugViewRecord

Examples

 for(metadata in pubchem_kit::resolve_repository("./repo/")) {
     metadata = metadata.pugView(metadata);
     
     # processing on the metabolite metadata
     # .......
 }
