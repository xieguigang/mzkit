pugView {pubchem_kit} R Documentation

query pubchem data via a given cid value

Description

Usage

pugView(cid,
    cacheFolder = "./pubchem_cache",
    offline = FALSE,
    sleep = 2);

Arguments

cid

-

cacheFolder

A cache directory path to the pubchem xml files. [as string]

sleep

sleep task in time interval seconds if no cache hit. [as double]

offline

[as boolean]

env

[as Environment]

Details

an attribute data which its named hit_cache is attached into the result object. which indicates that the current query is hit the cache or not.

Authors

BioNovoGene

Value

A collection of the pubchem pug view object that contains the metabolite annotation information.

clr value class
