mzpack {mzPack} R Documentation

open a mzpack data object reader, not read all data into memory in one time.

Description

Usage

mzpack(file);

Arguments

file

the file path for the mzpack file or the mzpack data object it self

env

[as Environment]

Details

a in-memory reader wrapper will be created if the given file object is a in-memory mzpack object itself

Authors

BioNovoGene

Value

the ms scan data can be load into memory in lazy require by a given scan id of the target ms1 scan

clr value class

  • IMzPackReader
