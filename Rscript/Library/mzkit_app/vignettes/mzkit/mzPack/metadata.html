metadata {mzPack} R Documentation

get metadata list from a specific ms1 scan

Description

Usage

metadata(mzpack, index);

Arguments

mzpack

A mzpack data lazy reader object that created via mzpack function.

index

the scan id of the target ms1 scan data

Details

Authors

Value

this function returns data object of type list.

clr value class

  • list

Examples

 let rawdata = open.mzpack(file = "./rawdata.mzPack");
 let scan_id = mzPack::ls(mzpack = rawdata);
 
 print(scan_id);
 
 for(id in scan_id) {
     str(mzPack::metadata(rawdata, id));
 }
