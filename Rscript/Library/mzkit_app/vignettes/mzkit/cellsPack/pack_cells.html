pack_cells {cellsPack} R Documentation

pack of the multiple raw data files into one data pack

Description

Usage

pack_cells(rawdata,
    source.tag = NULL);

Arguments

rawdata

a character vector of the raw data file path.

source.tag

usually be the organism source name. [as string]

env

[as Environment]

Details

this function is different with the function pack_cells.group, the rawdata file used at here is already been a sample pack, each scan inside the sample pack is a single cell.

Authors

BioNovoGene

Value

this function returns data object of type any kind.
