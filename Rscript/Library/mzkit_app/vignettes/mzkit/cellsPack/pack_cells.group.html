pack_cells.group {cellsPack} R Documentation

pack of the single cells metabolomics data in multiple sample groups

Description

Usage

pack_cells.group(groups,
    source.tag = NULL,
    verbose = FALSE);

Arguments

groups

could be a character vector of the folder path of the raw data files, it is recommended that using a tuple list for set this sample group value, the key name in the tuple list is the sample group name and the corresponding value is the folder path of the single cells rawdata files.

source.tag

usually be the organism source name. [as string]

env

[as Environment]

Details

this function required of each single rawdata file just contains only one single cell data.

Authors

BioNovoGene

Value

this function returns data object of type any kind.

clr value class

  • any kind

Examples

 require(mzkit);

 # set single cells sample data folders
 # each folder is one sample
 let single_cell_source = ["/datafiles/bulk_group_1"
 "/datafiles/bulk_group_2"
 "/datafiles/bulk_group_3"];

 # create single cells rawdata pack
 single_cell_source 
 |> pack_cells.group(tag = "Saccharomyces_cerevisiae")
 |> write.mzPack(file = "/datafiles/Saccharomyces_cerevisiae.mzPack")
 ;
