open.matrix {SingleCells} R Documentation

open a single cell data matrix reader

Description

Usage

open.matrix(file);

Arguments

file

-

env

[as Environment]

Details

this function open a lazy reader of the matrix, for load all data into memory at once, use the read.mz_matrix function.

Authors

BioNovoGene

Value

A tuple list that contains the data elements: 1. tolerance: the mass tolerance description for seperates the ion features 2. featureSize: the number of the ion features in the raw data file 3. ionSet: a numeric vector of the ion features m/z value. 4. spots: the number of the spots that read from the rawdata matrix file 5. reader: the rawdata MatrixReader

the list data also has some specificied data fields: list(tolerance, featureSize, ionSet, spots, reader).

clr value class

  • list

Examples

 let reader = open.matrix("/file.dat");
 let matrix = read.mz_matrix(reader$reader);
