as.expression {SingleCells} R Documentation

Cast the ion feature matrix as the GCModeller expression matrix object

Description

Usage

as.expression(x,
    single.cell = FALSE);

Arguments

x

[as MzMatrix]

Details

Authors

BioNovoGene

Value

the gcmodeller expression matrix object, each DataFrameRow element inside the generated matrix object is the expression vector of all metabolite ion features. which means the matrix format from this function outputs should be: 1. cell labels, or spatial location in rows 2. and ion features in columns.

clr value class

  • Matrix

Examples

 as.expression(x);
