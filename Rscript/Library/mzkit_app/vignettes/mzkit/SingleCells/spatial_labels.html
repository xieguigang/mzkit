spatial_labels {SingleCells} R Documentation

get the labels based on the spatial information of each spot

Description

Usage

spatial_labels(x);

Arguments

x

[as MzMatrix]

Details

Authors

BioNovoGene

Value

this function returns data object of type string.

clr value class

  • string

Examples

 let matrix = read.mz_matrix("/file.dat");
 let xyz = spatial_labels(matrix);
 let spatial = strsplit(xyz, ",");
 
 # x
 print(spatial@{1});
 # y
 print(spatial@{2});
 # z
 print(spatial@{3});
