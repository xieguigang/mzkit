norm_msdata {math} R Documentation

normalized the peak intensity data, do [0,1] scaled.

Description

Usage

norm_msdata(msdata,
    sum = FALSE);

Arguments

msdata

Should be a collection of the LibraryMatrix object.

sum

the intensity normalization method to be used in this function, set this parameter value to TRUE means use the total ion normalization method, and the default value FALSE means used the max intensity value for normalize the intensity value to a relative percentage value. [as boolean]

env

[as Environment]

Details

Authors

BioNovoGene

Value

A collection of the LibraryMatrix object with the intensity value for each ms2 peak normalized.
