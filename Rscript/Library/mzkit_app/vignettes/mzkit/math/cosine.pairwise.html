cosine.pairwise {math} R Documentation

pairwise alignment of the spectrum peak set

Description

Usage

cosine.pairwise(query, ref,
    tolerance = "da:0.3",
    intocutoff = 0.05);

Arguments

query

a spectrum set of the sample query input.

ref

a spectrum set of the reference library

tolerance

the ion m/z mass tolerance value for make the peak alignment

intocutoff

spectrum peak cutoff by relative intensity. [as double]

env

[as Environment]
