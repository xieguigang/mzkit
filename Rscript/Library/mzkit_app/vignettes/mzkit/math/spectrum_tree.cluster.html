spectrum_tree.cluster {math} R Documentation

create spectrum tree cluster based on the spectrum to spectrum similarity comparison.

Usage

spectrum_tree.cluster(ms2list,
    compares = NULL,
    tolerance = "da:0.1",
    intocutoff = 0.05,
    showReport = FALSE);
