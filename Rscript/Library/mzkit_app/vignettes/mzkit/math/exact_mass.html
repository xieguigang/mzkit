exact_mass {math} R Documentation

evaluate all exact mass for all known precursor type.

Description

Usage

exact_mass(mz,
    mode = "+");

Arguments

mz

a single ion m/z value. [as double]

mode

the ion polarity mode +/- for evaluate all kind of precursor type under the specific polarity mode, or a vector of the MzCalculator precursor type model which is generates from the math::precursor_types function.

Details

Authors

BioNovoGene

Value

a collection of the exact mass evaluation result, could be cast to dataframe via as.data.frame function.
