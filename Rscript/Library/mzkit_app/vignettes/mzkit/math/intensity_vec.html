intensity_vec {math} R Documentation

Extract an intensity vector based on a given peak index

Description

Usage

intensity_vec(ms, mzSet);

Arguments

ms

-

mzSet

A peak index object, which could be generated based on a given set of the peak m/z vector via the function mz_index. [as MzPool]

Details

Authors

BioNovoGene

Value

the returns value of this function is based on the input `ms` data: 1. for a single msdata object, then this function just returns a intensity numeric vector 2. for a collection of the msdata object, then this function will returns a dataframe object that each row is the element corresponding in the input msdata collection and each column is the m/z peak intensity value across the input msdata collection.

clr value class

  • double
