pixelId {MSI} R Documentation

get pixels [x,y] tags collection for a specific ion

Description

Usage

pixelId(raw, mz,
    tolerance = "da:0.1");

Arguments

raw

a ms-imaging rawdata object in mzpack format.

mz

a m/z numeric value

tolerance

the mass tolerance error for match the ion in the rawdata.

env

-

Details

Authors

Value

a character vector of the pixel [x,y] tags.

clr value class

  • string

Examples

 let rawdata = open.mzpack("/path/to/msi_rawdata.mzPack");
 let xy = MSI::pixelId(rawdata, mz = 100.0023, tolerance = "da:0.01");
 
 print(xy);
