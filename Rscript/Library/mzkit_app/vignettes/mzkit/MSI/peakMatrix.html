peakMatrix {MSI} R Documentation

Extract the ion data matrix

Description

Usage

peakMatrix(raw,
    topN = 3,
    mzError = "da:0.05",
    ionSet = NULL,
    raw.matrix = FALSE);

Arguments

raw

[as mzPack]

topN

select top N ion feature in each spot and then union the ion features as the features set, this parameter only works when the `ionSet` parameter is empty or null. [as integer]

ionSet

A tuple list of the ion dataset range, the tuple list object should be in data format of [unique_id => mz]. Or this parameter value could also be a numeric vector of the target m/z feature values

mzError

The mass tolerance of the ion m/z

env

[as Environment]

Details

feature elements inside the generated matrix object keeps the same order with the input ion features.

Authors

BioNovoGene

Value

the data format of the two kind of the output data result is keeps the same: + for a raw matrix object, the column is the ion features and the rows is the spatial spots. + for a dataset collection vector, the column is also the ion features and the rows is the spatial spots.

clr value class

  • MzMatrix
  • DataSet

Examples

 let raw = open.mzpack("/path/to/rawdata.mzPack");
 let ionsSet = list(ion1 = 100.0321, ion2 = 563.2254, ion3 = 336.9588);
 
 MSI::peakMatrix(raw, ionSet = ionsSet, mzError = "da:0.05");
