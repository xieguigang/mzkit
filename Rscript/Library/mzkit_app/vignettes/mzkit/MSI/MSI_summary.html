MSI_summary {MSI} R Documentation

Fetch MSI summary data

Description

Usage

MSI_summary(raw,
    x = NULL,
    y = NULL,
    as.vector = FALSE,
    dims = NULL);

Arguments

raw

[as mzPack]

as.vector

returns the raw vector of iPixelIntensity if set this parameter value to value TRUE, or its wrapper object MSISummary if set this parameter value to FALSE by default. [as boolean]

dims

overrides the MSI data its scan dimension value? This parameter value is a numeric vector with two integer element that represents the dimension of the MSI data(width and height)

Details

Authors

BioNovoGene

Value

this function returns data object in these one of the listed data types: MSISummary, iPixelIntensity.

clr value class

  • MSISummary
  • iPixelIntensity

Examples

 let rawdata = open.mzpack("/path/to/rawdata.mzPack");
 let spots = read.csv("/path/to/region.csv");
 let into = MSI_summary(rawdata, x = as.integer(spots$x), 
       y = as.integer(spots$y), 
       as.vector = TRUE);
 
 print(as.data.frame(into));
 print("view of the intensity vector:");
 print([into]::totalIon);
