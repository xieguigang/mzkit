MSI_summary {MSI} R Documentation

Fetch MSI summary data

Description

Usage

MSI_summary(raw,
    x = NULL,
    y = NULL,
    as.vector = FALSE,
    dims = NULL);

Arguments

raw

-

as.vector

returns the raw vector of @T:BioNovoGene.Analytical.MassSpectrometry.Assembly.MarkupData.imzML.iPixelIntensity if set this
parameter value to value TRUE, or its wrapper object @T:BioNovoGene.Analytical.MassSpectrometry.Assembly.MarkupData.imzML.MSISummary
if set this parameter value to FALSE by default.

dims

overrides the MSI data its scan dimension value? This parameter value is
a numeric vector with two integer element that represents the dimension
of the MSI data(width and height)

Details

Value

this function returns data object in these one of the listed data types: MSISummary, iPixelIntensity.

Examples

let rawdata = open.mzpack("/path/to/rawdata.mzPack");
 let spots = read.csv("/path/to/region.csv");
 let into = MSI_summary(rawdata, x = as.integer(spots$x), y = as.integer(spots$y), as.vector = TRUE);
 
 print(as.data.frame(into));
 print("view of the intensity vector:");
 print([into]::totalIon);
