correction {MSI} R Documentation

calculate the X axis scale

Description

Usage

correction(totalTime, pixels,
    hasMs2 = FALSE);

Arguments

totalTime

the max rt of the y scan data. [as double]

pixels

the average pixels of all your y scan data. [as integer]

hasMs2

does the ms-imaging raw data contains any ms scan data in ms2 level?. [as boolean]

Details

Authors

BioNovoGene

Value

A x axis correction function wrapper, the clr object type of this
function return value is determined based on the flag parameter
hasMs2:

  1. for has ms2 data inside your ms-imaging rawdata, a ScanMs2Correction object should be used,
  2. for has no ms2 data, a ScanTimeCorrection object is used
    for run x axis correction based on the average rt diff.

clr value class

  • Correction

Examples

 # create x-axis correction object based on the row scans data
 let cor = MSI::correction(totalTime = 120, pixels = 300, 
      hasMs2 = FALSE);
