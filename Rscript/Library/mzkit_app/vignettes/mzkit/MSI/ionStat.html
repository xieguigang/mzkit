ionStat {MSI} R Documentation

Extract the ion features inside a MSI raw data slide sample file

Description

Usage

ionStat(raw,
    grid.size = 5,
    da = 0.01,
    parallel = TRUE);

Arguments

raw

the raw data object could be a mzpack data object or MS-imaging ion feature layers object

grid.size

the grid cell size for evaluate the pixel density. [as integer]

da

the mass tolerance value, only works when the input raw data object is mzpack object. [as double]

Details

count pixels/density/etc for each ions m/z data

Authors

BioNovoGene

Value

this function returns data object of type IonStat.

clr value class

  • IonStat

Examples

 let rawdata = open.mzpack("/file.mzpack");
 let ion_features = MSI::ionStat(rawdata);
 
 print(as.data.frame(ion_features));
