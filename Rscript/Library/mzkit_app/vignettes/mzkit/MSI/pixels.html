pixels {MSI} R Documentation

get pixels size from the raw data file

Description

Usage

pixels(file,
    count = FALSE);

Arguments

file

imML/mzPack, or a single ion layer of the ms-imaging rawdata

count

get the pixel count number instead of get the canvas dimension size of the ms-imaging. [as boolean]

Details

Authors

BioNovoGene

Value

this function will returns the pixels in dimension size(a tuple list data with slot keys w and h) if the count is set to FALSE, by default; otherwise this function will return an integer value for indicates the real pixel counts number if the count parameter is set to TRUE.

the list data also has some specificied data fields: list(w, h).

clr value class

  • list

Examples

 let layer = MSI::as.layer(x, context = 100.0013);
 let counts = MSI::pixels(layer, count = TRUE);
 let canvas_size = MSI::pixels(layer, count = FALSE);
 
 print("the data pixels of current ion layer:");
 print(counts);
 print("the canvas dimension size of the ms-imaging:");
 str(canvas_size);
 print("the pixel number of the ms-imaging canvas:");
 print(canvas_size$w * canvas_size$h);
