raster {MSI} R Documentation

cast the ms-imaging layer data to raster object

Description


use this function for cast raster object, for do spatial heatmap rendering in another method.

Usage

raster(x,
    layer = Total);

Arguments

x

-

layer

the layer type for create the raster object, this parameter only works for when the data type of `x` is MSISummary. [as IntensitySummary]

env

[as Environment]

Details

Authors

BioNovoGene

Value

this function returns data object of type RasterMatrix.
