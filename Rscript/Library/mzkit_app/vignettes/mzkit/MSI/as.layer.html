as.layer {MSI} R Documentation

cast the pixel collection to a ion imaging layer data

Description

Usage

as.layer(x,
    context = "MSIlayer",
    dims = NULL,
    strict = TRUE);

Arguments

x

Should be a collection of the ms-imaging pixel data object, or a mz matrix object

context

the ms-imaging layer title, must be a valid mz numeric value if the input x is a mz matrix object

dims

the dimension size of the ms-imaging layer data, this dimension size will be evaluated based on the input pixel collection data if this parameter leaves blank(or NULL) by default.

strict

if the input **`dims`** produce invalid dimension size value, example as dimension size is equals to ZERO [0,0], then in strict mode, the dimension value will be evaluated from the input raw data automatically for ensure that the dimension size of the generated layer data object is not empty. [as boolean]

env

[as Environment]

Details

Authors

BioNovoGene

Value

A ms-imaging layer object that could be used for run ms-imaging rendering.

clr value class

  • SingleIonLayer

Examples

 imports "SingleCells" from "mzkit";
 imports "MsImaging" from "mzplot";
 
 let msi = SingleCells::read.mz_matrix("/path/to/msi_matrix.mat");
 let layer = MSI::as.layer(x, context = 100.0013);
 
 bitmap(file = "/path/to/save.png") {
     plot(layer);
 }
