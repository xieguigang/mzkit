splice {MSI} R Documentation

split the raw 2D MSI data into multiple parts with given parts

Description

Usage

splice(raw,
    partition = 5);

Arguments

raw

the mzpack rawdata object that used for run ms-imaging data analysis. [as mzPack]

partition

this parameter indicates that how many blocks that will be splice into parts on width dimension and column dimension. the number of the pixels in each partition block will be evaluated from this parameter. [as integer]

Details

Authors

BioNovoGene

Value

A collection of the ms-imaging mzpack object that split from multiple parts based on the input `raw` data mzpack inputs.

clr value class

  • mzPack

Examples

 let rawdata = open.mzpack("/path/to/msi_rawdata.mzPack");
 let parts = MSI::splice(rawdata, partition = 6);
