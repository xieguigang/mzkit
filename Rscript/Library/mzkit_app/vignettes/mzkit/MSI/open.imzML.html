open.imzML {MSI} R Documentation

open the reader for the imzML ms-imaging file

Description

Usage

open.imzML(file);

Arguments

file

the file path to the specific imzML metadata file for load for run ms-imaging analysis. [as string]

env

[as Environment]

Details

Authors

BioNovoGene

Value

this function returns a tuple list object that contains 2 slot elements inside: 1. scans: is the [x,y] spatial scans data: ScanData. 2. ibd: is the binary data reader wrapper object for the corresponding ibd file of the given input imzML file: ibdReader. 3. metadata: get file imzMLMetadata from the imzML header.

the list data also has some specificied data fields: list(scans, ibd, metadata).

clr value class

  • list

Examples

 # the msi_rawdata.ibd file should be in the same folder with the input imzml file.
 let imzml = open.imzML(file = "/path/to/msi_rawdata.imzML");
 
 # view metadata
 print(as.data.frame(imzml$metadata));
