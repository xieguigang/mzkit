candidates {formula} R Documentation

find all of the candidate chemical formulas by a

Description

specific exact mass value and a specific mass
tolerance value in ppm

Usage

candidates(mass,
    ppm = 5,
    ... = NULL);

Arguments

mass

the exact mass value

ppm

the mass tolerance value in ppm

candidateElements

a list configuration of the formula candidates

Details

Value

this function returns data object of type FormulaComposition[].

Examples

[Package formula version 1.0.0.0 Index]
