eval {formula} R Documentation

Evaluate formula exact mass

Description


evaluate exact mass for the given formula strings.

Usage

eval(formula);

Arguments

formula

a vector of the character formulas.

Details

-1 will be return if the given formula is empty or error/invalid

Authors

BioNovoGene

Value

the result data type is keeps the same as the given data type of the `formula` parameter: this function returns a numeric vector if the given `formula` is a character vector, or this function returns a key-value pair tuple list if the given `formula` is a list.

clr value class

  • double
