as.mzpack {mzweb} R Documentation

convert assembly file to mzpack format data object

Description

Usage

as.mzpack(assembly,
    ... = NULL);

Arguments

assembly

-

args

1. modtime: [GCxGC] the modulation time of the chromatographic run. modulation period in time unit 'seconds'. 2. sample_rate: [GCxGC] the sampling rate of the equipment. If sam_rate is missing, the sampling rate is calculated by the dividing 1 by the difference of two adjacent scan time. 3. imzml: [MS-Imaging] the pixel spot scan metadata collection for read ms data from the ibd file, the corresponding assembly object should be a ibdReader object 4. dims: [MS-Imaging] the canvas dimension size value for the ms-imaging heatmap rendering. [as list]
