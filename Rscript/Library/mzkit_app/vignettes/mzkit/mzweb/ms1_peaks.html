ms1_peaks {mzweb} R Documentation

get a overview ms1 spectrum data from the mzpack raw data

Description

Usage

ms1_peaks(mzpack,
    tolerance = "da:0.001",
    cutoff = 0.05,
    ionset = NULL);

Arguments

mzpack

The mzpack rawdata object. [as mzPack]

tolerance

The mass tolerance error

cutoff

intensity cutoff percentage value for removes the noised liked peaks. [as double]

ionset

A numeric vector for make subset of the ion features which is extract from the input mzpack rawdata file object.

env

[as Environment]

Details

Authors

BioNovoGene

Value

A ms peaks object

clr value class

  • LibraryMatrix

Examples

 let rawdata = open.mzpack("/path/to/rawdata.mzPack");
 let ms1 = ms1_peaks(rawdata, tolerance = "da:0.001", 
      cutoff = 0.01, 
      ionset = [347.56 238.3712 631.31 131.23]);
