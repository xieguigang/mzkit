setThumbnail {mzweb} R Documentation

set thumbnail image to the raw data file

Description

Usage

setThumbnail(mzpack, thumb);

Arguments

mzpack

[as mzPack]

thumb

Thumbnail image object data can be a gdi+ image or bitmap or a gdi+ canvas object in type ImageData. And also this parameter could be a lambda function that could be used for invoke and generates the image data.

Details

the parameter value of the `thumb` lambda function will be `mzpack` parameter value input

Authors

BioNovoGene

Value

returns a modified mzpack data object with Thumbnail property data has been updated.
