open_mzpack.xml {mzweb} R Documentation

open mzpack data from a raw data file in xml file format.

Description

Usage

open_mzpack.xml(file,
    prefer = NULL,
    da = 0.001,
    noise.cutoff = 0.0001,
    verbose = TRUE);

Arguments

file

the file path to the xml rawdata file. [as string]

prefer

the prefer file format used when the given `file` its extension suffix name is XML. value of this parameter could be imzml/mzml/mzxml. [as string]
