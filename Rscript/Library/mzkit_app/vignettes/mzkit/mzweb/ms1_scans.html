ms1_scans {mzweb} R Documentation

get all ms1 scan data points

Description

Usage

ms1_scans(mzpack,
    mz = NULL,
    tolerance = "ppm:20");

Arguments

mzpack

[as mzPack]

Details

Authors

BioNovoGene

Value

this function returns data object of type ms1_scan.

clr value class

  • ms1_scan

Examples

 let rawdata = open.mzpack(file = "./rawdata.mzPack");
 let ms1 = rawdata |> ms1_scans();
 
 # get xic liked data
 let xic = rawdata |> ms1_scans(mz = 999.0911, tolerance = "da:0.01");
 let mz_vec = [xic]::mz;
 
 print(as.data.frame(xic));
 
 bitmap(file = "./mz_histogram.png") {
     plot(hist(mz_vec));
 }
