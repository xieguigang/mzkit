parse_base64 {mzweb} R Documentation

Parse the given network base64 data as spectrum

Description

Usage

parse_base64(mz, intensity,
    id = NULL,
    auto.scalar = TRUE);

Arguments

mz

a character vector of the base64 string for the ms2 spectrum ion peaks

intensity

a character vector of the base64 string for the corresponding ion peaks intensity value.

id

a character vector of the spectrum reference id

auto.scalar

returns a scalar spectrum object if the input data just contains one spectrum data or not?. [as boolean]

env

[as Environment]
