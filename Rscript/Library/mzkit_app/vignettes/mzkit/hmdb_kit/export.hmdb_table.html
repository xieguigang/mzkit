export.hmdb_table {hmdb_kit} R Documentation

save the hmdb database as a csv table file

Description

Usage

export.hmdb_table(hmdb,
    file = NULL);

Arguments

hmdb

A collection of the HMDB metabolite. [as pipeline]

file

this function will returns a huge metabolite table if this parameter value default null

env

[as Environment]

Details

Authors

BioNovoGene

Value

this function returns the data depends of the `file` parameter is existsed or not: for file parameter has been omit, then a vector of the hmdb MetaDb clr object will be returns, otherwise a logical value for indicates the write table file success or not will be returns.

clr value class

  • boolean
  • MetaDb
