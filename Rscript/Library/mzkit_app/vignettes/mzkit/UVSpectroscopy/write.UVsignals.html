write.UVsignals {UVSpectroscopy} R Documentation

write UV signal data into a text file or netCDF4 data file

Description

Usage

write.UVsignals(signals, file,
    enable.CDFextension = FALSE);

Arguments

signals

a vector or pipeline of GeneralSignal

file

the file path of the data file that will be write signal data to it. [as string]

enable.CDFextension

only available for sciBASIC.NET product when this option is set to TRUE. not supports for the standard netcdf library on linux platform or other cdf file reader software like NASA Panoply, etc. [as boolean]

env
