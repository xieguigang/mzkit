parallel {spectrumTree} R Documentation

Enables or disables parallel processing for spectral searches.

Description

Usage

parallel(search, enable);

Arguments

search

The PackAlignment object to configure.

enable

TRUE to enable parallel processing, FALSE otherwise. [as boolean]

Details

this function only works for the PackAlignment method.

Authors

BioNovoGene

Value

The modified PackAlignment object.

  • any kind
