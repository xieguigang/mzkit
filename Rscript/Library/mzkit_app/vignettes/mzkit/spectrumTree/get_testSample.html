get_testSample {spectrumTree} R Documentation

Extract the test sample data for run evaluation of the annotation workflow

Description

Usage

get_testSample(packlib,
    n = 30,
    rtmax = 840,
    source.name = "get_testSample");

Arguments

packlib

The SpectrumReader object containing reference spectra. [as SpectrumReader]

n

The number of test samples to generate. [as integer]

rtmax

The maximum retention time for generated test spectra. [as double]

source.name

A fake source name for label this generated test dataset. [as string]

Details

Authors

BioNovoGene

Value

this function returns data object of type list. the list data also has some specificied data fields: list(peaktable, rawdata).

clr value class

  • list

Examples

 let libfile = spectrumTree::readpack(file = "./hmdb_lib.pack");
 let [peaktable, rawdata] = spectrumTree::get_testSample(libfile, 
     source_name = "hmdb metabolites test dataset");
 
 print(as.data.frame(peaktable));
 
 write.mzPack(rawdata, file = "./datafile.mzPack");
