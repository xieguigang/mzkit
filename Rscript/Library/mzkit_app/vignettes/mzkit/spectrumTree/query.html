query {spectrumTree} R Documentation

do spectrum family alignment via cos similarity

Description

Usage

query(tree, x,
    maxdepth = 1024,
    treeSearch = FALSE,
    top.hits = 3);

Arguments

tree

The reference spectrum tree object to search. [as Ms2Search]

x

The query spectrum data from the sample raw data files

maxdepth

The max depth of the tree search. [as integer]

treeSearch

Do alignment in family tree search mode?. [as boolean]

top.hits

the top n hits of the candidate result populated for the each query input, set this parameter value to zero or negative value means no limits. [as integer]

env

[as Environment]
