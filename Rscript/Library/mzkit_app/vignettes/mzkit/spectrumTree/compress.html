compress {spectrumTree} R Documentation

Compresses and optimizes a spectrum library, removing redundant entries.

Description

Usage

compress(spectrumLib, file, metadb,
    nspec = 5,
    xrefDb = NULL,
    test = -1);

Arguments

spectrumLib

The SpectrumReader source library. [as SpectrumReader]

file

Output file path for compressed library.

metadb

Metadata repository for annotations. [as IMetaDb]

nspec

Maximum spectra per metabolite entry. [as integer]

xrefDb

Cross-reference database name. [as string]

test

Number of test entries to process (-1 for all). [as integer]

env

The R environment for error handling. [as Environment]
