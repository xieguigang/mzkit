embedding {spectrumTree} R Documentation

Generates vector embeddings for spectral data (e.g., for machine learning).

Description

Usage

embedding(x,
    mslevel = 2);

Arguments

x

Input spectra (mzPack, PeakMs2 array, or mzPack list).

mslevel

MS level for spectra extraction (1 or 2). only works when the input dataset is mzPack. [as integer]

env

The R environment for error handling. [as Environment]

A VectorModel containing spectral embeddings.
