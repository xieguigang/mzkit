new {spectrumTree} R Documentation

Creates a new reference spectrum database file for storing spectral data.

Description

Usage

new(file,
    type = Default);

Arguments

file

The file path where the spectrum database will be saved.

type

The type of cluster structure to create (Pack, Binary, or Tree). [as ClusterTypes]

env

The R environment for error handling and output. [as Environment]

Details

Authors

BioNovoGene

Value

An instance of ReferenceTree, ReferenceBinaryTree, or SpectrumPack based on the specified type.

clr value class

  • ReferenceTree
  • ReferenceBinaryTree
  • SpectrumPack

Examples

 libfile <- spectrumTree::new(file = "./hmdb_lib.pack", type = "Pack");
