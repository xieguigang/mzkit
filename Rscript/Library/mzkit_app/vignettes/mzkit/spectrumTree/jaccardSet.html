jaccardSet {spectrumTree} R Documentation

construct a fragment set library for run spectrum search in jaccard index matches method

Description

Usage

jaccardSet(libname, mz, mzset,
    rt = NULL,
    cutoff = 0.1,
    filter.complex.adducts = FALSE);

Arguments

libname

Array of library spectrum identifiers. [as string]

mz

Array of precursor m/z values. [as double]

mzset

Array of fragment m/z strings (comma-separated). [as string]

rt

Array of retention times (optional). [as double]

cutoff

Jaccard similarity threshold (0.0 to 1.0). [as double]

filter.complex.adducts

Exclude spectra with multiple adducts if TRUE. [as boolean]

env

The R environment for error handling. [as Environment]

Value

A JaccardSearch object for fragment-based searches.
