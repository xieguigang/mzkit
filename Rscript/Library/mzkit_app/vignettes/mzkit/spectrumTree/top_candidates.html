top_candidates {spectrumTree} R Documentation

Retrieves the top candidate matches from a metabolite library search.

Description

Usage

top_candidates(libs, x,
    top = 9);

Arguments

libs

The metabolite library (Library(Of MetaLib)). [as Library`1]

x

The query spectrum (PeakMs2 or GCMSPeak).

top

The maximum number of top candidates to return. [as integer]

Details

Authors

BioNovoGene

Value

An array of AlignmentOutput objects representing top matches.

clr value class

  • AlignmentOutput
