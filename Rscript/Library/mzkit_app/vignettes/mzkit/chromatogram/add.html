add {chromatogram} R Documentation

Add a chromatogram data in the chromatogram overlap collection

Description

Usage

add(overlaps, name, data);

Arguments

overlaps

A chromatogram overlap collection object to be add new layer to it

name

usually be a sample name

data

usually be a chromatogram data that extract from a sample data

Details

Value

this function returns data object of type ChromatogramOverlap.

clr value class

  • ChromatogramOverlap
