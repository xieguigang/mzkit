toChromatogram {chromatogram} R Documentation

Convert chromatogram tick point data to a chromatogram object

Description

Usage

toChromatogram(ticks,
    name = NULL);

Arguments

ticks

A vector of the ChromatogramTick object or a dataframe object. If the data input is a dataframe object, then the data fields is required for create the chromatogram object: 1. rt,RT,retention_time: used for represents the retention time data, data should be in time data unit seconds 2. basePeak,BPC: used for represents the base peak ion intensity value 3. totalIon,TIC: used for represents the total ions intensity value 4. into,intensity: used for represents the ion intensity value if the basePeak or total ion data is not specific

Details

Authors

BioNovoGene

Value

A chromatogram data object, which could be used for do chromatogram overlaps or data plot visualization.
