checkInSourceFragments {annotation} R Documentation

Check the ms1 parent ion is generated via the in-source fragment or not

Description

Usage

checkInSourceFragments(ms1, ms2,
    da = 0.1,
    rt.win = 5);

Arguments

ms1

the ms1 peaktable dataset, it could be a xcms peaktable object dataframe, a collection of ms1 scan with unique id tagged.

ms2

the ms2 products list

env

[as Environment]

Details

Authors

BioNovoGene

Value

a tuple key-value pair list object that contains the flags for each ms1 ion corresponding slot value TRUE means the key ion is a possible in-source fragment ion data, otherwise slot value FALSE means not.

clr value class

  • boolean
