peak_assign {annotation} R Documentation

Associates the ms1 peaks with the ms2 spectrum alignment result hits.

Description

Usage

peak_assign(works, libname, adducts, xcms.id, mz, rt, RI, npeaks);

Arguments

works

the workspace object, which could be constructed via the workspace function. [as LibraryWorkspace]

libname

the reference key to the ms2 spectrum alignment result. [as string]

adducts

[as string]

xcms.id

[as string]

mz

[as double]

rt

[as double]

RI

[as double]

npeaks

[as integer]

Details

a ms2 spectrum alignment result should be existed inside the workspace before assign the ms1 peaks to the result.

Authors

BioNovoGene

Value

This function has no value returns.
