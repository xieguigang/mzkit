XIC {data} R Documentation

get chromatogram data for a specific metabolite with given m/z from the ms1 scans data.

Description

Usage

XIC(ms1, mz,
    tolerance = "ppm:20");

Arguments

ms1

a sequence data of ms1 scans

mz

target mz value. [as double]

tolerance

tolerance value in unit ppm or da for
extract m/z data from the given ms1 ion
scans.

env

[as Environment]

Details

Authors

BioNovoGene

Value

this function returns data object in these one of the listed data types: ms1_scan, ChromatogramTick, ChromatogramOverlap.

clr value class

  • ms1_scan
  • ChromatogramTick
  • ChromatogramOverlap
