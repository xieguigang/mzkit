XIC {data} R Documentation

Extracts chromatogram data for a specific m/z from MS1 scans.

Description

Usage

XIC(ms1, mz,
    tolerance = "ppm:20");

Arguments

ms1

Input MS1 data (mzPack, PeakSet, or scan array).

mz

Target m/z value to extract. [as double]

tolerance

Mass tolerance for m/z matching.

env

R environment for error handling. [as Environment]

Details

Authors

BioNovoGene

Value

ChromatogramTick array or ChromatogramOverlap object.

clr value class

  • ms1_scan
  • ChromatogramTick
  • ChromatogramOverlap
