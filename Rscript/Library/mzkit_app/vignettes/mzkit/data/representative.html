representative {data} R Documentation

Generates a representative spectrum by aggregating (sum or mean) input spectra.

Description

Usage

representative(x,
    mean = TRUE,
    centroid = 0.1,
    intocutoff = 0.05);

Arguments

x

Input spectra (PeakMs2 or LibraryMatrix collection).

mean

If true, uses mean intensity; otherwise sums intensities. [as boolean]

centroid

Mass tolerance for centroiding peaks. [as double]

intocutoff

Relative intensity cutoff (0-1) to filter weak peaks. [as double]

env

R environment for error handling. [as Environment]
