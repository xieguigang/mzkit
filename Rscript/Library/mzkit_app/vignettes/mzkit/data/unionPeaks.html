unionPeaks {data} R Documentation

Union and merge the given multiple spectrum data into one single spectrum

Description

Usage

unionPeaks(peaks,
    matrix = FALSE,
    massDiff = 0.1);

Arguments

peaks

A collection of the spectrum object that going to merge into single one. [as PeakMs2]

matrix

this parameter will affects the data type of the value returns of this function: 1. default false, returns a peak ms2 data object 2. true, returns a library matrix data object. [as boolean]

massDiff

the mass error for merge two spectra peak. [as double]

Details

Authors

BioNovoGene

Value

a single ms spectrum data object, its data type depeneds on the `matrix` parameter.

clr value class
