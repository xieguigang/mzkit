alignment_ref {data} R Documentation

get alignment result tuple: query and reference

Description

Usage

alignment_ref(align,
    query = "Query",
    reference = "Reference");

Arguments

align

AlignmentOutput object from spectral matching. [as AlignmentOutput]

query

Name label for query spectrum. [as string]

reference

Name label for reference spectrum. [as string]

Details

Authors

BioNovoGene

Value

a tuple list object that contains spectrum alignment result: 1. query - spectrum of sample query 2. reference - spectrum of library reference

the list data also has some specificied data fields: list(query, reference).

clr value class

  • list
