peakMs2 {data} R Documentation

Constructs a PeakMs2 object from spectral data.

Description

Usage

peakMs2(precursor, rt, mz, into,
    totalIons = 0,
    file = NULL,
    libname = NULL,
    precursor.type = NULL,
    ... = NULL);

Arguments

precursor

Precursor m/z value. [as double]

rt

Retention time in seconds. [as double]

mz

Array of fragment m/z values. [as double]

into

Array of fragment intensities. [as double]

totalIons

Total ion current (optional). [as double]

file

Source file identifier. [as string]

libname

Library identifier. [as string]

precursor.type

Precursor adduct type. [as string]

meta

Metadata list for the peak. [as list]

env

R environment for error handling. [as Environment]

A PeakMs2 object containing the spectral data.
