intensity {data} R Documentation

Extracts intensity values from MS1 scans or PeakMs2 spectra.

Description

Usage

intensity(ticks,
    mz = -1,
    mzdiff = "da:0.3");

Arguments

ticks

Input scans or spectra.

mz

Optional m/z to extract specific intensity. [as double]

mzdiff

Mass tolerance for m/z matching.

env

R environment for error handling. [as Environment]

Details

Value

Numeric vector of intensity values.
