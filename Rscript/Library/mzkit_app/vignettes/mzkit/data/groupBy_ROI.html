groupBy_ROI {data} R Documentation

make a tuple list via grouping of the spectrum data via the ROI id inside the metadata list

Description

Usage

groupBy_ROI(peakms2,
    default = "Not_Assigned");

Arguments

peakms2

a collection of the spectrum data to make spectrum data grouping.

default

the default ROI id for make the data groups if the metadata is null or the ROI id tag is missing from the spectrum object metadata. [as string]

env

[as Environment]

Details

Authors

BioNovoGene

Value

A tuple list object that contains the spectrum data grouping by the ROI id. The key of the tuple list is the ROI id, and the value is a list of spectrum data that belongs to this ROI id. If no ROI id was assigned, a warning message will be added to the runtime environment.

clr value class
