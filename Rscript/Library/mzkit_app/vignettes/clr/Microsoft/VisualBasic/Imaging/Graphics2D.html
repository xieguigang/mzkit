Graphics2D {Microsoft.VisualBasic.Imaging} .NET clr documentation

Graphics2D

Description

GDI+ device handle for encapsulates a GDI+ drawing surface.
(GDI+绘图设备句柄，这个对象其实是为了将gdi+绘图与图形模块的SVG绘图操作统一起来的)
