MummichogAnnotator

Description

Mummichog预注释器 基于KEGG通路先验知识, 在无MS/MS数据的情况下, 仅利用一级质谱m/z和统计p值进行代谢物预注释 核心思想: "先通路分析, 后代谢物注释" (Guilt-by-association)

Declare

            
# namespace BioNovoGene.BioDeep.MSEngine.Mummichog
export class MummichogAnnotator {
   # 获取加合物频率背景模型 (步骤4完成后可用)
   AdductFrequencies: list;
   # 获取所有匹配结果 (步骤3完成后可用)
   AllMatches: iterates(MzMatch);
   # 获取/设置算法参数
   Parameters: MummichogParams;
   # 获取通路富集分析结果 (步骤5完成后可用)
   PathwayResults: iterates(PathwayEnrichmentResult);
   # 获取理论m/z库 (步骤2完成后可用)
   TheoreticalLibrary: iterates(TheoreticalMz);
}
